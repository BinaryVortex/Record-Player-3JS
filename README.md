# Record Player 3JS

A 3D interactive record player simulation built with Three.js, featuring glowing light emitters and realistic belt-driven mechanics.

## Demo

![Screenshot](Screenshot%202026-04-13%20151943.png)

You can view the live demo by opening `index.html` in your browser.

## Features

- Interactive 3D record player model
- Rotating rollers and belt simulation
- Glowing light emitters along the spiral path
- Bloom post-processing effects
- Orbit controls for camera manipulation
- Shadow mapping for realistic lighting

## Technologies

- Three.js (r148)
- WebGL
- JavaScript ES Modules

## How to Run

1. Clone the repository
2. Open `index.html` in a modern web browser

No additional dependencies or build steps required.
